# AI-Powered Resume–Job Matching System

This project uses **Machine Learning and NLP** to match resumes with job descriptions and calculate a similarity score.

---

## Features
- Extracts text from PDF resumes
- Cleans and processes text
- Matches resume with job description
- Shows similarity score

---

## Technologies Used
- Python  
- Streamlit  
- Scikit-learn  
- NLTK  
- PyPDF2  

---

## Project Structure
```

├── app.py
├── requirements.txt
├── src/
│   ├── parser.py
│   ├── preprocessing.py
│   └── matcher.py

````

---

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
````
